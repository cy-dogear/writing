# <span id="c1">我的五项兴趣爱好</span>

/ 2025-11

兴趣爱好是：**没有必要做，但自主地喜欢做又可以经常做的事。**

人生需要兴趣爱好活动，这样活着才更有意思。

我来捋一捋自己现在有哪些兴趣爱好。

## 1. 写作

45岁以后，我才明确意识到自己爱好写作。之前，我写了近30年日记，写日记从不需要坚持，因为有话要说，写出来心里舒服些。不过我的日记不是每天写，有话说才写。

45岁以后，写作对于我来说不再只是有话要说，我开始追求表达的效率和美感，开始学习排版和作图、表，追求信息传播的效果。

我的写作不涉及虚构创作，论文、杂文、记叙文、诗歌是主要形式。

【输出形式】写作自带“输出”。

## 2. 读书

20岁之前我曾经爱读小说和散文，现在不爱读了，小说中某些不合情理的虚构令我出戏。现在我最爱读历史和哲学类。

【输出形式】不动笔墨不读书。我一定要写读后感，哪怕只写一两句话，也比不动笔要好。

## 3. 运动保健

凡是运动我都喜欢。连我已经放弃的游泳我也是喜欢的。我放弃游泳是因为它麻烦，不适合作为日常项目。

目前我最喜欢自重健身、静功调息（比如站桩）、三公里跑步、徒步、散步、篮球、乒乓球。

【输出形式】目前没有专题输出。有时会写点运动中的感想、日记。

## 4. 手工DIY

电子维修、旧衣改造、木工家具、旧房改造……凡是自己动手美化生活的事，我都喜欢。那是创造的乐趣，是变废为宝的欣喜。

自己做饭算不算手工DIY？我觉得算。我每天做饭。

【输出形式】自带输出，DIY作品就是输出。

## 5. 追求极简

这项爱好如何介绍？懂的都懂，那种不被束缚的轻松感，令我着迷。

【输出形式】目前没有专题输出。有时会写点记叙文作记录。

## 6. 这些活动不是我的爱好

| 我不喜欢 | 不能作爱好的首要原因 |
| --- | --- |
| 旅游 | 费心费钱 |
| 电子游戏 | 觉得它不健康，让人脱离现实 |
| 宠物 | 住在城市不要宠物，把宠物当儿子养的方式让我难受 |
| 美食聚餐 | 美食是好，可是不应沉迷，所以不作爱好 |
| 棋牌 | 长时间坐着多累啊，偶尔玩几局就好 |
