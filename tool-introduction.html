📌 "mini-MD" is a Minimal Markdown Editor

一、它是什么？与常规 Markdown 解析器的区别

1 定位：一个“20kB”的轻量级输入中转站，用于带简易标记的纯文本初稿写作。

我习惯用plain TXT写初稿，现在觉得在TXT基础上增加一些markdown标记，做到“纲举目张”也挺好，所以有了以下设计安排。
我不喜欢市面上的markdown编辑器（诸如notepad, VS code, WPS, Feishu），它们的解析设置不给用户修改（或是我没找到），所以做了这个叫做“mini-MD”的小工具。

2 与常规工具的核心不同：

  • 保留空行：您输入的回车空行会原样保留（常见解析器会压缩连续空行），便于纯文本环境下阅读。
  • 最小语法集：只支持 三级标题（#, ##, ###） + 无序列表（- 或 *），在这工具里用不着更多语法标记；这工具不是为了替代VS Code。
  • 标题层级映射特殊：# → 标题4，## → 标题5，### → 标题6 —— 这是为了后续粘贴到 Word 时能直接匹配我的文档中的层级（从4级开始）。
  • 极度轻量：约20KB的单HTML文件，免安装，跨平台（保存到桌面即可离线使用）。

二、怎么使用它？

  1. 在线访问：→ https://cy-dogear.github.io/writing/mini-MD.html
  2. 离线使用：将HTML文件保存到桌面，双击打开即可。
  3. 日常操作：
    • 写完想另起新篇 → 刷新浏览器，获得全新空白输入区。
    • 导出到其他软件 → 点击 “全选并复制”，然后粘贴到 Word、微信、知乎 等编辑器（选择 “保留源格式”），即可得到已排版的文本。

三、自定义修改（源代码就在HTML文件中）

  • 修改解析规则：找到约 第270行 的 parseMinimalMarkdown() 函数，改动输出格式（例如改变标题样式，实现粘贴到WORD以后你想要的格式）。
  • 修改快捷字符按钮：左栏上方有6个用于快速输入的字符按钮，想自定义它们，请修改约 第350行 的 quickChars 字符串数组。

📌 “frequent-char” 是写作时经常要用的字符串输入工具

一、它是什么？与常见同类软件的区别

1 定位：适配“个人使用频率”和“个性使用习惯”的特殊字符输入工具。

当我们要输入“⚠”“≠”这类特殊字符，或输入“闵行区报春路888弄……”这类长字符串时，通常有以下几种方式：

  • 输入法提供的自定义短语
  • 热键工具提供的字符串映射
  • 编辑器自带的特殊字符菜单（如字符映射表、Word符号菜单）
  • 码点输入触发
  • 互联网上搜索再复制
  • 自备TXT文件保存常用字符，复制粘贴使用

这些方法我统统觉得不好用：

  • 菜单窗口太小，翻找想用的字符太费劲
  • 从选取到完成输入，动作路径过长
  • 需要记忆映射编码，我有时记岔了，而且编码触发的有效概率不是100%

2 与常规工具的核心不同：

  • 适配“个人使用频率”：frequent-char能统计用户使用频率，把高频字符排在前面
  • 还可以手动修改频率排序，实现个性化插队占位
  • 适配“个性使用习惯”：特殊字符和常用短语不分类别、混排在一个大窗口里，反而更好找（频率排序后，不需要分类）
  • 每个字符单元可以是单独的“①”“②”，也可以合并成“①②③④”，随意安排
  • 点击一次，即复制对应字符
  • 跨平台，免安装，轻量化

二、怎么使用它？

  1. 在线访问：https://cy-dogear.github.io/writing/frequent-char.html
  2. 离线使用：将HTML文件保存到桌面，双击打开即可
  3. 日常操作：
    • 点击想用的字符单元，即完成复制
    • 想记录使用频率时，在窗口下方的表单区粘贴该字符，回车，使用频率“+1”被记录
  4. 自定义修改：窗口右下方的「导入」「导出」按钮可导出常用字符列表为TXT文件，修改TXT文件即可完成上述“个性使用习惯”适配……我懒得写更详细的说明啦。这个软件是我自己用的，我不从中谋利，懒得推广。分享到网上，希望有人喜欢它。
